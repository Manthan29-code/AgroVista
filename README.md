## AgroVista — React + Vite Prototype

Quick start

Requirements
- Node 18+

Commands

```
npm install
npm run dev
```

Build

```
npm run build
npm run preview
```

Tech
- React + Vite, Tailwind CSS, React Router
- Recharts (charts), React Leaflet (map)

Mock Data
- `src/mocks/data/*`

Demo Script
1. Register, go to Dashboard
2. Farm Input → submit
3. Recommendations → Compare
4. Pest Scan upload → modal
5. Market → Map toggle
6. Profile → Offline toggle shows banner
